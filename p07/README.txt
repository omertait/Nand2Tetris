****** VM Translator ******
by : Omer Taitelman & Gilad Twili
This Project contains VM Translator as part of project 7 of NandtoTetris course
(handle only arithmetic/push/pop commands)

*** Written in Java ***

Input : vm file path (written in VM commands)
Output : asm file in the same directory as input

To run the VM Translator:
1. compile - javac .\VMTranslator.java
2. run - java .\VMTranslator.java  xxx.vm (xxx - file path)
